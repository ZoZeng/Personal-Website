Getting Started¶

Before getting started, you will need to install the various libraries that we will use. We will be using many commonly-used libraries such as Pandas and Numpy, as well as Matplotlib and Ggplot for plotting data and Statsmodels for linear regression models and features. All of these canbe installed using pip:

% pip install

This tutorial will use data from Imdb pulled from Kaggle.com. The database is accessed through URL in this tutorial, but the database can also, be The database can also be downloaded into the folder of this notebook to be accessed locally on your machine through https://www.kaggle.com/orgesleka/imdbmovies.

Bivariate Analysis¶

We will now perform bivariate analysis for the data. We'll train a linear regression model with Statsmodels for each variable of interest against 'imdbRating' with the hopes of uncovering what effect these variables have on 'imdbRating'.

In [352]:
reg = smf.ols(formula='imdbRating ~ year', data=imdb).fit()
b,m = reg.params
x = imdb['year']
imdb.plot.scatter(x='year',y='imdbRating',s=.02)
plt.plot(x,m*x + b,'-',color='y')
Out[352]:
[<matplotlib.lines.Line2D at 0x4ab43210>]

The scatterplot shows that 'year' has a negative correlation with 'imdbRating'. This means that ratings will be increasingly harsher as the years go on. The opaqueness of the right side of the scatterplot shows a large majority of movies are younger. This corresponds with what we've seen from the histogram for 'year'.

We will now just summary() to on our trained regression model to reveal more information about the distribution. This analysis procedure will be used for all variables of interest.
